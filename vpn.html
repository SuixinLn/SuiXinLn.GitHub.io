小白入门级阿里云VPN搭建教程-文字版


阿里云轻量服务器购买地址：

    https://www.aliyun.com/product/swas?spm=5176.19720258.J_3207526240.35.5e052c4apiZs5s

支持的操作系统 （CentOS 7+、Ubuntu 16+、Debian 8+），操作流程是一样的。在第3步申请SSL证书的时候，请输入对应系统的命令，其他的是一样的。

服务器购买后，搜索DNS加速，复制服务器的公网IP地址，与域名进行解析。

如果外网网页打开速度慢，无法下载应用等问题，可联系随心Ln索要免费临时节点使用。

    1) 安装finalshell，下载安装连接：

    https://www.hostbuf.com/


2) 开启防火墙
        iptables -P INPUT ACCEPT
        iptables -P FORWARD ACCEPT
        iptables -P OUTPUT ACCEPT
    iptables -F
        apt-get purge netfilter-persistent
 reboot 重新开始


    3) 申请SSL证书

        apt update -y       # Debian/Ubuntu 命令
        apt install -y curl   # Debian/Ubuntu 命令
        apt install -y socat  # Debian/Ubuntu 命令
        yum update -y        #CentOS 命令
        yum install -y curl    #CentOS 命令
        yum install -y socat   #CentOS 命令

       curl https://get.acme.sh | sh

        ~/.acme.sh/acme.sh --register-account -m landxin@qq.com

 #将邮箱地址更改为你自己的


        ~/.acme.sh/acme.sh --issue -d www.baidu.com --standalone #将”www.baidu.com” 更换为你自己绑定的域名地址
#改变你的解析域



        ~/.acme.sh/acme.sh --installcert -d www.baidu.com --key-file /root/private.key --fullchain-file /root/cert.crt

    #将”www.baidu.com” 更换为你自己绑定的域名地址

    #更换你的解析域名，此步完成后会在VPS root目录下；看到证书公钥/root/cert.crt及验证文件/root/private.key


    4) 安装X-ui面板

        bash <(curl -Ls https://raw.githubusercontent.com/vaxilu/x-ui/master/install.sh)


    5) 安装最新版V2ray，下载链接：
    https://github.com/2dust/v2rayN/releases

    6) 如果打不开v2rayN.exe,可以安装.NET Framework 4.8，下载链接：
    https://docs.microsoft.com/zh-cn/dotnet/framework/install/guide-for-developers
